Setup

Running in under 5 minutes

Three steps. No Docker required to start (though it's supported).

1

Install nanobot

One command. Works on any machine with Python ≥ 3.11.

# Recommended (uv)
uv tool install nanobot-ai

# Or with pip
pip install nanobot-ai
2

Configure your channels + LLM

Run onboard to get a guided setup. Add your API key and channel bot token.

nanobot onboard

# ~/.nanobot/config.json
"model": "deepseek/deepseek-chat",
"channels": ["telegram"]
3

Start your bot

Your assistant is live on every enabled channel. Invite it to a group or DM it directly.

nanobot start

# → Connected: Telegram ✓
# → Connected: Discord ✓
# → Memory: active
Installation

Pick your install method

# Install uv first if you don't have it
curl -LsSf https://astral.sh/uv/install.sh | sh

# Install nanobot
uv tool install nanobot-ai

# Onboard (guided setup)
nanobot onboard

# Start
nanobot start
pip install nanobot-ai

nanobot onboard
nanobot start
# Run onboard in Docker to generate config
docker run -v ~/.nanobot:/root/.nanobot \
--rm nanobot onboard

# Edit config (add your API keys)
vim ~/.nanobot/config.json

# Run gateway
docker run -v ~/.nanobot:/root/.nanobot \
-p 2525:2525 nanobot start
git clone https://github.com/HKUDS/nanobot.git
cd nanobot
pip install -e .

nanobot onboard
nanobot start
