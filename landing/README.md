# LightRAG Landing Page

Production-ready single-file landing page for [LightRAG](https://github.com/HKUDS/LightRAG).

## Deploy

**Vercel** — drag-and-drop the folder at [vercel.com/new](https://vercel.com/new), select "Other" as framework preset.

**GitHub Pages** — enable Pages on the repo, source: root `/`.

**Local preview** — `open index.html` or `npx serve .`

## Custom domain

After deployment, set `lightrag.io` as your custom domain in the platform settings and add the corresponding DNS records (see parent README for A-record values).
