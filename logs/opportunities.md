# 机会日志 · Opportunity Log

> 记录所有发现的行动机会，用状态标签跟踪进展。
> 规则：**每一个洞察都必须能导向至少一个具体行动**。

---

## 使用方法

每条记录用以下格式：

```
## [状态] 仓库/领域: 机会标题

- **发现于**: YYYY-MM-DD
- **来源**: GitHub动态 / arXiv论文 / 用户反馈 / 自主发现
- **维度**: 技术 / 市场 / 人员 / 时机 / 行动
- **优先级**: A/B/C/D

**背景**: 3-5 句话描述背景

**洞察**: 我看到了什么别人没看到的？

**具体行动**:
- [ ] 行动 1
- [ ] 行动 2

**预期结果**: 如果行动成功，会发生什么？

---
```

状态标签: `🔵新发现` `🟡进行中` `🟢已完成` `🔴已放弃`
