## CEO 的 TOP 10 押注清单

> 如果只能用 12 个月，资源有限，按此顺序分配：

1. **LightRAG** → 核心引擎，一切变现的基础设施层。优先级：商业化 API 服务
2. **MiniRAG** → LightRAG 的边缘/隐私版，打不同市场细分。优先级：中小企业

4. **RAG-Anything** → LightRAG 的多模态扩展包，捆绑销售
5. **nanobot** → 最快 PMF，最低成本，最宽渠道覆盖
6. **FastCode + FastAgent** → 开发者工具，打 MCP/Cursor 生态位
7. **DeepCode** → 学术背书 + 机构 R&D 定向销售
8. **VideoRAG** → 媒体/教育 B2B，差异化赛道
9. **Paper2Slides** → 最低工程成本，最直接面向科研用户
10. **LightReasoner + SepLLM** → AI 效率工具包，卖给 AI 中小企业
